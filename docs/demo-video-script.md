# Demo Video Script

Target length: 50-70 seconds.

## Slide 1 - Problem

Builders lose time deciding which bounties or hackathons are worth doing. Paid Bounty Triage Agent packages that first decision into a paid, callable agent service.

## Slide 2 - Callable Agent Loop

A requester submits a bounty or hackathon URL. The agent returns priority, token estimate, cash probability, risks, and a delivery checklist.

## Slide 3 - Payment and Receipt

The demo creates a paid call envelope, mock USDC settlement, CAP-like lifecycle events, and an auditable receipt. Real wallet signing is a manual gate.

## Slide 4 - Verified Run

Show the generated HTML summary and receipt evidence from `artifacts/latest-http-demo`.

## Slide 5 - Real CROO Path

Replace `MockSettlementAdapter` with CROO SDK methods: `negotiate_order`, `accept_negotiation`, `pay_order`, `deliver_order`, and `get_delivery`.

## Slide 6 - Submission Package

Public repo, README, architecture diagram, rules check, submission copy, mock receipt, screenshot, and release artifact are ready.
